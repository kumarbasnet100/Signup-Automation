## Project Overview
This project automates the signup flow of The Authorized Partner portal using Playwright.
It includes email OTP verification using MailSlurp and file upload validation.

---

## Tech Stack
- Language: TypeScript
- Automation Framework: Playwright
- Test Runner: Playwright Test
- Email Service: MailSlurp
- Environment Management: dotenv
- OS Tested: macOS

---

## Prerequisites
- Node.js (v18 or above)
- npm
- Git
- MailSlurp API Key

---
